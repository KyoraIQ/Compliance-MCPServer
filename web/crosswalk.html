The crosswalk

One risk, every framework's answer

A crosswalk shows how the same underlying risk is addressed across different frameworks. Pick a risk below and see which control in each framework covers it.

Each card is a risk that lives in one layer of your stack. The pills underneath are the controls — across different frameworks — that address that same risk. This is what makes a control in one framework findable from any other.
