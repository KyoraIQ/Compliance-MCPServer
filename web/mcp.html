What is MCP, and how do I connect?

MCP (Model Context Protocol) is a standard way to give an AI assistant a set of tools it can use. Connect your assistant to Kyora IQ and it can look up real compliance controls instead of guessing.

The idea in one minute

Normally an AI assistant only knows what it learned during training. An MCP server hands it a menu of tools it can call. Once your assistant is connected to Kyora IQ, you can ask it things like "What does NIST AC-2 require, and what does it map to in HIPAA?" and it will call our tools to get the real, cited answer.

The URL is the address

It tells your assistant where the server lives. Knowing an address alone doesn't let anyone in.

The token is the key

Only assistants with the key can use the tools. Your instructor gives you the key privately.

This server is read only and contains no secrets. It only returns the public compliance information you can already see on this site. Keep your access token private, do not post it publicly, and it cannot be abused.

Connect Claude

1Open Claude's settings and find Connectors (or "Custom connectors").
2Add a connector using the server URL your instructor gave you.
3When prompted, provide your access token as a bearer token.
4Start a chat: "Using Kyora IQ, search for audit logging controls."

Menu names and supported plans change over time. If you don't see Connectors, check Claude's current documentation.

Generate your connection details

Paste the URL and token your instructor gave you. This runs entirely in your browser. Nothing is sent anywhere or stored, and the token is never shown on this public page by default.
